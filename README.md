# Weekly Fiat Report

This repository publishes the weekly fiat on/off-ramp product research report.

## Published Report

- Latest report: <https://nobodydufer-droid.github.io/weekly-fiat-report/reports/latest.html>
- Historical reports are stored under `reports/`.

## Structure

- `index.html`: redirects to the latest report.
- `reports/latest.html`: current published report.
- `reports/每周法币出入金产品能力调研-YYYY-MM-DD.html`: archived weekly reports.
